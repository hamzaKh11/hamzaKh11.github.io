Projet d'Analyse de Donnes : prdire la survie des passagers durant le voyage du Titanic

Introduction¶

1. problme

Le 15 avril 1912, au cours de son voyage inaugural, le Titanic a coul aprs avoir heurt un iceberg, tuant 1502 passagers et membres dquipage sur 2224. Taux de survie traduit de 32 %. Lune des raisons pour lesquelles le naufrage a caus de telles pertes de vie, cest quil ny avait pas suffisamment dembarcations de sauvetage pour les passagers et lquipage. Bien quil y ait eu un certain facteur de chance dans la survie au naufrage, certains groupes de personnes taient plus susceptibles de survivre que dautres, comme les femmes, les enfants et la classe suprieure. Ainsi, le voyage du Titanic a gnr beaucoup de donnes que les statisticiens ont collect fin de les analyser et les mettre en valeur pour prdire d'autres situations et viter la rptition d'un tel problme .

2. Le data set

Le data set est compos de 12 colonnes et 418 lignes o 7 colonnes sont quantitatives et 5 qualitatives, il a comme valeurs nulles 414 rpartis entre plusieurs colonnes

Une brève description de chaque colonne est la suivante :¶

PassengerId - ID unique d'identification du passager Survived - Drapeau de survie (0 = mort, 1 = survie) Pclass - Classe de billet Name - Nom du passager Sex - Genre (homme = homme, femme = femme) Age - Age SibSp - Nombre de frres et surs / conjoints bord du Titanic Parch - Nombre de parents / enfants sur le Titanic Ticket - Numro de ticket Fare - tarif Cabin - Numro de chambre Embarked - Point de dpart (port sur Titanic)

Nous allons galement donner une brve description de chaque variable. pclass = classe de billet 1 = classe suprieure (riche) 2 = classe intermdiaire (classe gnrale) 3 = classe infrieure (classe ouvrire)

Embarked = La dfinition de chaque variable est la suivante C = Cherbourg Q = Queenstown S = Southampton NaN = reprsente une perte de donnes.

3. Hypothses et questions

Je me souviens peine quand j'ai regard le film Titanic pour la premire fois, mais Titanic reste toujours un sujet de discussion dans les domaines les plus divers. Ainsi plusieurs questions se pose ce points :

  • Quel genre de peuple survit ?
  • Quels sont les facteurs influenant la survie des personnes ?
  • L'age des personnes survis ?
  • Quelle classe dominant les classes des survivants ?
  • Peut on dire que les femmes et les enfants ont une forte chance survivre ?
